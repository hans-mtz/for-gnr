vmlclearerrstatus

Sets the VM Error Status to VML_STATUS_OK and stores the previous VM Error Status to olderr.

Syntax

olderr = vmlclearerrstatus( )

Include Files

  • mkl_vml.f90

Output Parameters

Name

Type

Description

olderr

INTEGER

Specifies the former VM error status.
