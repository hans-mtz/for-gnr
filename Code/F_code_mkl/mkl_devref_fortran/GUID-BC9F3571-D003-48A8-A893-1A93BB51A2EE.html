?larfg

Generates an elementary reflector (Householder matrix).

Syntax

call slarfg( n, alpha, x, incx, tau )

call dlarfg( n, alpha, x, incx, tau )

call clarfg( n, alpha, x, incx, tau )

call zlarfg( n, alpha, x, incx, tau )
