Intel® oneAPI Math Kernel Library Developer Reference - Fortran

v?Pow2o3

Computes the cube root of the square of each vector element.

Syntax

call vspow2o3( n, a, y )

call vspow2o3i(n, a, inca, y, incy)

call vmspow2o3( n, a, y, mode )

call vmspow2o3i(n, a, inca, y, incy, mode)

call vdpow2o3( n, a, y )

call vdpow2o3i(n, a, inca, y, incy)

call vmdpow2o3( n, a, y, mode )

call vmdpow2o3i(n, a, inca, y, incy, mode)

Include Files

  • mkl_vml.f90

Input Parameters

Name

Type

Description

n

INTEGER, INTENT(IN)

Specifies the number of elements to be calculated.

a

DOUBLE PRECISION for vdpow2o3, vmdpow2o3

REAL, INTENT(IN) for vspow2o3, vmspow2o3

DOUBLE PRECISION, INTENT(IN) for vdpow2o3, vmdpow2o3

Arrays, specify the input vector a.

inca, incy

INTEGER, INTENT(IN)

Specifies increments for the elements of a and y.

mode

INTEGER(KIND=8), INTENT(IN)

Overrides global VM mode setting for this function call. See vmlSetMode for possible values and their description.

Output Parameters

Name

Type

Description

y

REAL, INTENT(OUT) for vspow2o3, vmspow2o3

DOUBLE PRECISION, INTENT(OUT) for vdpow2o3, vmdpow2o3

Array, specifies the output vector y.

Description

The v?Pow2o3 function computes the cube root of the square of each vector element.
